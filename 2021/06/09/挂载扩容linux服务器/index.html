挂载扩容 Linux 服务器

需求

  • Linux 服务器硬盘从 300G 扩容到 500G
  • /etc/fstab 持久性挂载

一、确认当前磁盘状态

使用 df -h 命令确认目前的空间状态。

二、确认新磁盘

使用 lsblk 命令来确认新磁盘是否被正确识别——发现新磁盘被正常识别到。

可以发现 /dev/vdb 是新分区。

使用 fdisk 命令确认磁盘信息。

三、创建文件系统

确认文件系统，发现新分区没有文件系统。
